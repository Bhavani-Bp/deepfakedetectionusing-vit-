## 🟩 SLICE 1 — Project Initialization & Environment Setup

**Objective:**
Prepare a stable, reproducible development environment.

**Includes:**

* Google Colab GPU enablement
* Python & dependency setup
* Folder structure creation
* Version locking (requirements.txt)

**Deliverables:**

* GPU verified
* All libraries import without error
* Project directory tree created

**Why first:**
Nothing else can proceed without a stable environment.

---

## 🟩 SLICE 2 — Dataset Acquisition & Organization

**Objective:**
Get a **working subset** of deepfake datasets ready for processing.

**Includes:**

* Dataset selection (FaceForensics++ / DFDC subset)
* Download & extraction
* Class separation (real vs fake)
* Storage optimization for Colab

**Deliverables:**

* `data/real/` and `data/fake/`
* At least 500–1000 samples total
* Dataset integrity verified

**Why now:**
Model design depends on data shape and quality.

---

## 🟩 SLICE 3 — Video Processing & Face Extraction

**Objective:**
Convert raw videos into **model-ready face images**.

**Includes:**

* Frame sampling from videos
* Face detection (MTCNN or RetinaFace)
* Face alignment & cropping
* Saving faces as images

**Deliverables:**

* Face images extracted correctly
* No broken frames
* Sample visual verification

**Why critical:**
Deepfake artifacts are facial — wrong extraction = bad model.

---

## 🟩 SLICE 4 — Dataset Loader & Augmentation Pipeline

**Objective:**
Prepare data for efficient and robust training.

**Includes:**

* PyTorch Dataset & DataLoader
* Image normalization
* Augmentations (flip, blur, compression)
* Train/validation split

**Deliverables:**

* DataLoader returns correct tensors
* Batch loading works on GPU
* No shape or dtype errors

**Why now:**
Training depends on a clean data pipeline.

## 🟩 SLICE 5 — Model Architecture Design

**Includes:**

* Backbone selection (EfficientNet / Xception)
* Final classification head
* Weight initialization

**Deliverables:**

* Model forward pass works
* Output shape correct

**Why isolated:**
Model definition should be correct before training begins.

## 🟩 SLICE 6 — Training Pipeline Implementation

**Includes:**

**Deliverables:**

* Training runs without crashing
* Validation accuracy ≥ ~85–90%
* Logs show convergence

**Why now:**
This is the **core intelligence** of the system.
